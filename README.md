# motion-driven-artificial-eye-system

Motion-Driven Artificial Eye System
This repository contains the final working Python code for my dissertation project:
Investigating Human Responses to Motion-Driven Gaze Behaviour in an Artificial Eye System
Project overview
This project explored whether a simplified artificial eye system, driven by motion detection, could create perceived attentiveness, a sensation of being watched, and mild unease in human observers.
The system uses webcam-based motion detection to control the gaze direction of a pair of artificial eyes displayed on screen in real time.
Main features
webcam-based motion detection using OpenCV
real-time visual display using Pygame
directional gaze mapping
smoothing to reduce jitter
directional lag to reduce rapid switching
idle scanning behaviour when no motion is detected
fullscreen pair-of-eyes display
optional video recording in the participant testing version
Software used
Python
OpenCV
Pygame
Blender (used to create the eye image assets)
How to run
To run the program, Python and the required libraries must be installed.
Required Python libraries:
opencv-python
pygame
Run the script in Python after making sure the eye image files are in the same folder as the code.
Notes
This repository contains the final working version of the prototype used for the project.
The system was developed as part of a final-year dissertation in Engineering with Physiology at the University of Reading.
